# Férias ESF - iOS

Projeto iOS simples com WKWebView para abrir:

https://dtechbr.github.io/gestao-ferias-esf/

## Dados do app

- Nome exibido: Férias ESF
- Bundle ID: com.dtech.feriasesf
- iOS mínimo: 15.0
- Tecnologia: Swift + WKWebView

## Como usar no GitHub Actions

1. Crie um repositório no GitHub.
2. Envie todos os arquivos e pastas deste ZIP para o repositório.
3. Entre na aba **Actions**.
4. Clique em **Build iOS**.
5. Clique em **Run workflow**.
6. Ao finalizar, baixe o arquivo em **Artifacts**.

Observação: o workflow compila para simulador iOS sem assinatura. Para gerar IPA instalável no iPhone, é necessário certificado Apple Developer e provisioning profile.
